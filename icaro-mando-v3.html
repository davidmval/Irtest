💡 Cómo funciona: CH340G a 76800 baud. El byte 0x55 (01010101) genera onda cuadrada 38.4kHz. Alternando 0x55 (carrier ON) y 0xFF (carrier OFF) con timing NEC/Samsung se generan señales IR reales. Cada frame son ~500 bytes enviados de golpe por USB bulk transfer.

Cómo Funciona

El problema: Las TVs esperan IR modulada a 38kHz. El CH340G solo envía bytes serie.

La solución:

1. CH340G a 76800 baud → cada bit = 13µs

2. Byte 0x55 = 01010101 → onda 38.4kHz ✓

3. Byte 0xFF = silencio (no carrier)

Protocolo NEC (32 bits):

• Header: 9ms carrier + 4.5ms silencio

• Bit 0: 560µs carrier + 560µs silencio

• Bit 1: 560µs carrier + 1690µs silencio

Protocolo Samsung (32 bits):

• Header: 4.5ms carrier + 4.5ms silencio

• Bits: igual que NEC

Frame completo: ~500 bytes enviados por USB en un solo bulk transfer. Se repite 2 veces con 40ms de gap.

💡 Verificar emisión: El LED IR (940nm) es invisible. Míralo por la cámara frontal de otro móvil — se verá como luz violeta parpadeando.
